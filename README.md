# React Mini Projects

This repository contains **20 mini React projects** demonstrating core React concepts like `useState`, `useEffect`, `Context API`, routing, forms, API fetching, and more. All projects are combined into **one React app** for easy navigation.

🔗 **Live Demo:** [React Mini Projects on GitHub Pages](https://sujan-roy24.github.io/react-mini-projects/)

---

## Projects Included

| #  | Project Name                  | Description |
|----|--------------------------------|-------------|
| 1  | Counter App                   | A simple counter using `useState`. Increment, Decrement & Reset with a button click. |
| 2  | Todo List                     | Add and remove tasks dynamically. Demonstrates array state management. |
| 3  | Digital Clock                 | A live clock using `useEffect` with interval updates. |
| 4  | Weather Card                  | Fetch weather data from OpenWeatherMap API and display it. |
| 5  | Form Validation               | Signup form with validation for email and password. |
| 6  | Image Gallery                 | Grid gallery using props and reusable image components. |
| 7  | Accordion FAQ                 | Expandable FAQ sections using conditional rendering. |
| 8  | Simple Router App             | Multi-page routing with Home, About, and Contact pages. |
| 9  | Context Theme Switcher        | Light/dark theme toggle using React Context API. |
| 10 | Shopping Cart                 | Add/remove items with total price calculation using `useReducer`. |
| 11 | Infinite Scroll               | Loads more items as you scroll using Intersection Observer. |
| 12 | Custom Hook Timer             | Reusable countdown timer hook with start, pause, and reset. |
| 13 | Modal Popup                   | Reusable modal using React Portals. |
| 14 | Redux Counter                 | Counter app managed with Redux and React Redux. |
| 15 | Styled Components Button      | Customizable buttons with CSS-in-JS using styled-components. |
| 16 | Error Boundary App            | Catch errors in child components with error boundaries. |
| 17 | Unit Tested Component         | Button component tested with Jest and React Testing Library. |
| 18 | Lazy Loading Images           | Lazy load image components for better performance. |
| 19 | GraphQL Todo                  | Fetch and mutate todos from a GraphQL API using Apollo Client. |
| 20 | Portfolio Site                | Personal portfolio site integrating multiple React concepts. |
